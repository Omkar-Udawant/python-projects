# Higher Lower Game (Python)

An interactive command-line game built with Python where players guess which public figure has more social media followers.

---

## About the Game

The game displays two famous personalities at a time.  
Your task is to decide who has a higher follower count.

- Guess correctly to increase your score  
- One wrong guess ends the game  
- The game continues as long as you keep winning  

---

## Features

- Randomized comparisons every round  
- Score tracking system  
- Clean and engaging CLI interface  
- ASCII art for better user experience  
- Beginner-friendly Python logic
