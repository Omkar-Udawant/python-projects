# Flight Deals Project

## Overview
This project is designed to find the best flight deals using various APIs like Sheety, Amadeus, and Twilio. It fetches flight data, checks for cheaper flights, and can send notifications via WhatsApp when a good deal is found.

## Features
- **Flight Search:** Uses the Amadeus API to search for flights based on given criteria.
- **Price Comparison:** Compares current flight prices to previously recorded prices.
- **Notifications:** Sends a WhatsApp message if a cheaper flight is found.
