# Stock News Alert

A Python project that tracks daily stock price changes and fetches related news when volatility exceeds a threshold.

## Features
- Fetches stock data using Alpha Vantage API
- Calculates daily percentage change
- Fetches top 3 related news articles
- Console-based alert output

## Tech Stack
- Python
- Requests
- Alpha Vantage API
- NewsAPI

## How to Run
1. Install dependencies
2. Add your API keys
3. Run `main.py`
