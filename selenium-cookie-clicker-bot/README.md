# Selenium Cookie Clicker Bot

This project is a simple automation script using Selenium WebDriver to play the Cookie Clicker game automatically.

## How to Run

1. Make sure you have Google Chrome and the appropriate ChromeDriver installed.
2. Install the Selenium package via pip if you haven't: `pip install selenium`
3. Run the script. The bot will start clicking the cookie and buying upgrades automatically.

## Features

- Automatically clicks the cookie and purchases the most expensive available upgrades.
- Runs for a specified amount of time and then exits, printing the final cookies-per-second score.

Feel free to customize or extend it as needed!
